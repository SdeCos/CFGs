# SimHub CFGs

Configuraciones para SimHub, incluyendo mapeos de controles y dashboards.

## Control Mapper
Se pueden importar desde la interfaz de SimHub o copiar manualmente.

**Ruta manual:** `C:\Program Files (x86)\SimHub\PluginsData\Common\`

- [x] [Config global](./ControlMapper/outputmapping.shomapping) - Roles y salidas globales.
- [x] [Config VRS DFP](./ControlMapper/VRS%20DirectForce%20Pro%20Formula%20Wheel%20[USB].shcmapping) - Mapeo específico para VRS DFP.
- [x] [Config Simucube 2](./ControlMapper/Simucube%202%20Pro.shcmapping) - Mapeo específico para Ascher B24/Simucube 2.

## Dashboards
Para instalar, haz doble clic en el archivo `.simhubdash`.
